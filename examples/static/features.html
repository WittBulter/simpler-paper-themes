Features

simpler-paper contains a lot of conventions, and in some cases you need only a few simple steps to achieve more customizing effects

Multilayer menu

if you want to create a menu that contains parent-child relationships, you just need to create folders. simpler-paper automatically generate menus based on hierarchical relationships.
like: 

**doc source**
|____docs
| |____install
| | |____install.1.md
| | |____install.2.md
| |____problems
| | |____answer.1.md
| | |____answer.2.md
| |____paper.config.json

Sort

both files and folders support manually specifying weights, you just need to add a prefix to them. prefixes are automatically removed after compilation.
