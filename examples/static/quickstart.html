QuickStart

Simpler Paper is a very simple and easy to use the document generator. you only need to run a command, All of the Markdown will be automatically integrated into the site.

Build and preview

with just one command, you can easily compile the file and the local preview.

# install 
npm install -g simpler-paper  

# init  
paper i

# don't forget to add some markdown files.  
# build to html
paper build 

# preview  
paper s

Deploy

# In default, the document will be released to github.    
paper deploy

set github pages:

  1. go to your rep.
  2. Settings > Options > GitHub Pages
  3. choose the source for gh-pages branch
