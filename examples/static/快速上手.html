快速上手

使用

  1. 引入 howl-theme

    "howl-theme": "git+ssh://git@github.com/eleme/howl-theme#x.x.xxxx"

  2. 绑定 (具体参见预览)

    import { Theme } from 'howl-theme'
Vue.use(Theme)

  3. 部分指令还携带有参数 

  import { Theme } from 'howl-theme'
  Vue.use(Theme, { options })

开发

  1. Install & Run

    npm i && make dev ${module}

  2. Build

    make build
